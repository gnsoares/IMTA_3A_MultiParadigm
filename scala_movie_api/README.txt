### Projet sbt compilé avec Scala 3

### Usage

Ceci est un projet sbt normal. Vous pouvez compiler le code avec `sbt compile`, l'exécuter avec `sbt run`, et `sbt console` démarrera un REPL Scala 3.

Avantages d'utilisation de objets fonctionnels:
* Code plus organisée
* Garantie de correct état de chaque objet
* Meilleure lisibilité
* On peut les utiliser comme clefs de Map
* Robustesse

Incovénients d'utilisation de objets fonctionnels:
* Code plus complexe
* Possiblement utilise plus de mémoire pour créer plusieurs objets
